# Altalena Web Server
## Version: 2025/11.26a (2025-11-26 17:40:00)

Altalena is a containerized Web Server built on Docker and Docker Compose.

It provides the complete LAMP stack:
- Apache Web Server;
- PHP 8.2;
- MySQL 8.0;
- phpMyAdmin (latest).

In order to start Altalena:

```console
altalena@server:~$ sudo ./start.sh
```

And to stop Altalena:

```console
altalena@server:~$ sudo ./stop.sh
```

Please notice that the scripts must be run as <b>sudo</b> (or root)!
